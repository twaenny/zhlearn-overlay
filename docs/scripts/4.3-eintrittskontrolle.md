## Bekannte Lücken (alle in OPEN-QUESTIONS.md #1)

- Plattform: iPad-native vs. responsive Web vs. Cross-Platform (iOS+Android)?
- Identifikation: QR-Code, Badge-Scan, manuelle Suche, Personalnummer?
- Offline: Sync-Queue + Indicator?
- Permissions: wer sieht Teilnehmerliste, Notes, GoShows?
- Retro-Booking-Trigger: GoShow → sofort SAP, oder erst local-Note + späteres Sync?
- Kiosk-Modus: dedicated Tablet vs. Kursleiter-Phone?
- App-User-Rollen: Externe Kursleitende dürfen GoShows erfassen oder nur intern?
