## Talking Points

- „**Sichtbarkeit + Terminierung auf einer Surface**: wer-sieht und wann gehören zusammen — Sichtbarkeit ohne Schedule ist unvollständig, Schedule ohne Sichtbarkeit sinnlos."
- „**Preview zeigt die echte Cohort** (`Sichtbar ab X für 3 Mandanten`) — nicht nur „scheduled", sondern die resultierende Audience."
- „**Vier-Augen-Freigabe** ist Kantonsstandard (Bid Beilage V §3.2.2). Routing:"
  - Normale Angebote → ZHlearn-Manager
  - Pflichtangebote + Angebote mit Kostenfolge → Superuser (nicht Manager)
  - Extern sichtbare Angebote → Admin (Staatskanzlei)
- „**3 Approval-Verben** (statt 2): `Mit Auflage freigeben` ist KTZH-typisch — approve aber „bitte Titel kürzen". Trade-off zu „approve / reject" simplicity bewusst akzeptiert."
- „**Status-Vokabular** (9 States lt. Bid §5):" Entwurf → In Prüfung → Korrektur erforderlich → Freigegeben → Geplant → Aktiv → Pausiert → Geschlossen → Archiviert. Wir zeigen Erstellt + Eingereicht + Freigegeben; die anderen verbal."
